คลิกปุ่มด้านล่างเพื่อดาวน์โหลดไอคอนทั้งสองขนาด แล้วบันทึกไว้ในโฟลเดอร์ icons/

ดาวน์โหลด icon-192.png ดาวน์โหลด icon-512.png

วิธีใช้งาน

  1. ดาวน์โหลดไฟล์ทั้งสอง
  2. วางไฟล์ไว้ในโฟลเดอร์ icons/ ของโปรเจกต์
  3. เปิด server.bat เพื่อรัน HTTP server
  4. เปิดเบราว์เซอร์ไปที่ http://localhost:8080
  5. บน Android: เมนู → "เพิ่มในหน้าจอหลัก" หรือ "ติดตั้งแอป"
